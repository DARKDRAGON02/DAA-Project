## 🧮 Implemented Programs

### 1️⃣ Dijkstra’s Algorithm

* Finds the shortest path from a source node to all other nodes in a graph.

### 2️⃣ Rod Cutting Problem

* Solves the problem of maximizing profit by cutting a rod into pieces.

### 3️⃣ Maximum Subarray Problem

* Finds the contiguous subarray with the maximum sum.
